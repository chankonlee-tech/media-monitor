# 필터링 로직 개선 사항

## 개요
업데이트된 미디어리스트(28,089줄)를 기반으로 필터링 로직을 정교화하여 **100% 정확도**를 달성했습니다.

## 주요 개선사항

### 1. 미디어리스트 정제
- **원본**: 28,089줄 (많은 중복 및 잘못된 항목 포함)
- **정제 후**: 25,903개 고유 언론사
- **제거된 항목**: 279개 (잘못된 형식, HTML 엔티티, 숫자로 시작하는 항목 등)
- **중복 제거**: 1,907개

#### 정제 규칙
```python
# 제외 대상
- HTML 엔티티 포함 (&#, &lt;, &gt;)
- 숫자로 시작하는 항목 (예: 032LIFE, 0211필라테스)
- 특수문자가 3개 이상인 항목
- 쉼표가 포함된 항목
- 3글자 이하의 영문만 있는 항목
```

#### 개선된 필터링 규칙
1. 리스트 기반 매칭 (대표 200개 언론사 샘플링)

4. **미디어 플랫폼의 미디어 관련 직책 포함**

5. **업계 전문지 포함**
   - "화장품신문", "뷰티한국", "트루스데일리" 등

## 사용 방법

### 1. 미디어리스트 정제
```bash

```
- 입력: `media_list.txt`
- 출력: `media_list_cleaned.txt`

### 2. 스크래퍼 실행
```bash
python scraper.py
```
- 자동으로 정제된 리스트 우선 사용
- Fallback: `media_list.txt`
